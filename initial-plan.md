# 自作のDNSフィルタ型広告回避システム構築
iPhone での web サイト閲覧時に、DNSフィルタレベルで回避可能な広告を
表示しないようにするシステムを構築してみます。

グローバルにアクセス可能なサーバーに wireguard の VPN を構築し、
その VPN 内で docker コンテナ内で動作する DNS を用意します。

oisd でしたっけ、広告フィルタ用のデータを用いて、該当パターンの DNS 解決要求を 0.0.0.0 に解決します。

フィルタ外のパターンはパブリック DNS にフォールバックさせます。

開発は Rust で行いたいです、既存の DNS 関連のライブラリがあったと思うのでそれを使用したいです。
最初はエントリの逆順文字列を比較する方式でフィルタ処理を行ってみたいです。
